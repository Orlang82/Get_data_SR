## Задача: формирование перечня сделок forex для 6S

**Цель:** получить перечень сделок forex с описанием по проведенным документам 6S.

**Входные данные:**
- Результат отработки скрипта fetch_pay_6sx_data, поле DESCRIPTION
- Результат парсинга поля DESCRIPTION с помощью скрипта parser_forex.py из папки utils

**Выходные данные:**
- Файл: SR_6SX_FOREX_template.sql - должен возвращать перечень forex сделок с описанием

**Первоначальные действия:**
1. получить результат отработки процедуры fetch_pay_6sx_data, поле DESCRIPTION
2. передать полученные данные в парсер parser_forex.py
3. передать полученные данные из парсера в запрос SR_6SX_FOREX_template.sql в переменную :data_number
5. можешь свободно редактировать скрипт parser_forex.py, подзадачу в случае необходимости, сохраняя только перечень возвращаемых полей

**Выходные данные:**
- резльтаты отработки запроса SR_6SX_FOREX_template.sql записываются в файл excel из котрого вызван скрипт в таблицу с именем t6S_FOREX, на листе 6SX_ACC - используй процедуру записи в таблицу paste_to_excel_smart

**Логирование:**
- допускается формирование логов, условие - логи в случае необходимости могут быть просто отключены

**Размещение:**
- скрипт долже быть в одном модуле с именем "forex_6sx" размещай в fetchers
- лог размещай в logs

**Рекомендации:**
- для понимания структуры данных прочитай файл SR_6SX_FOREX_template.sql и функцию парсера parser_forex.py
- скрипт должен быть компактным и быстрым, добавляй в код комментарии на русском
- в случае возникновения вопросов или уточнения обязательно спрашивай меня
