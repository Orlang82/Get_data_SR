# TASKS_chart_7s.md

## Задача: chart_7s

**Цель:** Создать скрипт построения графика «Динаміка мінімального розміру ринкового ризику» на основе данных из Excel-таблицы `tDB_History_2` и вставки готового изображения в рабочую книгу. График отображает динамику четырёх рядов данных (столбчатая гистограмма + три линии) за последние 12 наблюдений.

---

**Входные данные:**
- Таблица `tDB_History_2` на листе `sys` рабочей книги Excel (читается через xlwings):
  - `Дата` — дата наблюдения (datetime)
  - `МРРР` — значение МРРР в гривнях (ряд: столбчатая гистограмма)
  - `Валютний ризик` — значение валютного риска в гривнях (ряд: линия)
  - `Процентний ризик` — значение процентного риска в гривнях (ряд: линия)
  - `Товарний ризик` — значение товарного риска в гривнях (ряд: линия, вторичная ось)

---

**Первоначальные действия:**
1. Прочитать таблицу `tDB_History_2` с листа `sys` через xlwings API: `sheet.api.ListObjects("tDB_History_2")`
2. Преобразовать диапазон таблицы в `pandas.DataFrame` (заголовок — первая строка)
3. Убедиться, что колонка `Дата` имеет тип `datetime`, числовые колонки — `float`
4. Отобрать **последние 12 строк** по порядку (по колонке `Дата` или по позиции — строки уже отсортированы по дате)
5. Вычислить значения в тыс. грн. (делить каждое значение на 1 000; округлять до целых) — только для **подписей данных**, для построения графика использовать исходные значения в грн.

---

**Логика обработки:**

### Определение «разрыва» на столбцах гистограммы (МРРР)

Задача — не допустить, чтобы один аномально высокий столбец «прижал» остальные столбцы и линии к оси X.

1. Вычислить медиану значений `МРРР` по выбранным 12 точкам.
2. Если **хотя бы один** столбец превышает `МРРР_BREAK_MULTIPLIER × медиана` (параметр конфигурации, рекомендуемое значение: `2.5`) — активировать режим разрыва.
3. В режиме разрыва:
   - Ограничить визуальную высоту оси Y значением `y_cap`, которое вычисляется автоматически: `y_cap = медиана × МRRR_CAP_MULTIPLIER` (параметр конфигурации, рекомендуемое значение: `1.8`)
   - Для столбцов, чьё значение превышает `y_cap`, отрисовать столбец обрезанным по высоте `y_cap`
   - Поверх каждого обрезанного столбца нарисовать символ разрыва: две короткие косые полосы (∿∿), реализованные как два полигона `matplotlib.patches.FancyBboxPatch` или ручная отрисовка линий-«зигзагов» (`BREAK_LINE_HEIGHT`, `BREAK_LINE_WIDTH` — параметры конфигурации)
   - Подпись данных для обрезанных столбцов выводить **над символом разрыва**
4. Для столбцов без разрыва подпись данных выводить **над столбцом** стандартным образом.

### Форматирование подписей данных
- Значения для подписей = исходное значение / 1000, округлить до целых
- Формат: `"{:,.0f}".replace(",", " ")` — разделитель тысяч: пробел (украинский формат), без десятичных знаков
- Подписи выводить для **всех** рядов данных (МРРР, Валютний ризик, Процентний ризик, Товарний ризик)

---

**Выходные данные:**
- Изображение вставляется в лист `To_Report` рабочей книги, привязка к ячейке `C17`
- Размер изображения в Excel: `IMAGE_WIDTH × IMAGE_HEIGHT` пикселей (пропорция 17.5 : 7)
- Существующее изображение с именем `IMAGE_NAME` удаляется перед вставкой нового

**Структура скрипта:**

Рекомендуемое разделение на функции:

1. `get_chart_data() → pd.DataFrame` — читает таблицу `tDB_History_2`, возвращает последние `N_LAST` строк
2. `format_label(value_grn) → str` — форматирует значение в грн. → подпись в тыс. грн. с пробелом-разделителем
3. `draw_break_symbol(ax, x, y_cap, cfg)` — рисует символ разрыва на указанном столбце
4. `build_chart(df) → matplotlib.figure.Figure` — строит полную фигуру по данным; вся логика разрыва здесь
5. `save_chart(fig) → str` — сохраняет фигуру во временный файл, возвращает путь
6. `insert_chart_to_excel(img_path)` — удаляет старое изображение по `IMAGE_NAME`, вставляет новое в ячейку `IMAGE_CELL`
7. `create_market_risk_chart()` — **основная публичная функция**, последовательно вызывает 1→4→5→6, оборачивает в try/except с print-логированием
